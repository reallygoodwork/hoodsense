Hoodsort

How to Use

Parameters

Location: A string representing the neighbourhood you want to search.

Price: A string representing the max dollar amount you're willing to pay per month.

Rooms: A string representing the amount of rooms within the unit.

Hot Tip

The API takes quite some time since it scrapes Craigslist, visits each link and grabs the image. THEN, heads to Kijiji and repeats the process. I recommend using some sort of visual cue like a progress bar.
